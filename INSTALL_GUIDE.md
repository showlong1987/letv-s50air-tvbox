# 📺 乐视S50 Air TVBOX 安装完全指南

## 一、准备工作

### 确认电视信息
- 型号：乐视超级电视 S50 Air
- 系统：LetvUI 3.0（Android 4.3）
- 确认路径：设置 → 关于本机 → 查看系统版本

### 所需工具
- U盘（FAT32格式，≥4GB）
- 电脑一台
- 可选：手机安装「甲壳虫ADB助手」

---

## 二、开启ADB调试（两种路径）

### 方法A：工程菜单（推荐）
1. 打开电视，按遥控器 **信号源** 键
2. 输入数字 **2580**
3. 进入工程菜单 → 选择 **其他选项**
4. 滚动到底部 → 选择 **ADB调试** → 设为 **开启**
5. **重启电视**

### 方法B：开发者模式
1. 进入 **设置** → **关于本机**
2. 连续点击 **版本号** 7次
3. 返回设置 → 找到 **开发者选项**
4. 开启 **USB调试** / **ADB调试**

---

## 三、安装APK

### 方法一：U盘安装（最简单）
1. 将 `乐视TVBOX.apk` 复制到U盘根目录
2. 插入电视USB接口
3. 电视弹出提示 → 选择 **文件管理**
4. 找到APK文件 → 点击安装
5. 若提示"禁止未知来源"：
   - 设置 → 安全 → **允许安装未知来源应用** → 开启
   - 重新安装

### 方法二：ADB命令安装
```bash
# 1. 获取电视IP（设置 → 网络 → IP地址）
# 2. 连接电视
adb connect 192.168.1.100:5555   # 替换为你的电视IP

# 3. 安装APK
adb install -r 乐视TVBOX.apk

# 4. 启动应用
adb shell am start -n com.letv.tvbox/.LetvTVBoxActivity
```

### 方法三：甲壳虫ADB助手（手机端）
1. 手机和电视连接**同一WiFi**
2. 打开甲壳虫ADB助手
3. 自动搜索或手动输入电视IP
4. 连接后 → 选择 **安装APK** → 选择文件
5. 等待安装完成

## 四、首次使用配置

### 1. 打开应用
- 桌面会出现 **乐视TVBOX** 图标
- 遥控器点击即可打开

### 2. 配置接口源
- 按遥控器 **菜单键** 或点击右上角设置
- 在 **接口配置地址** 中输入接口URL
- 推荐接口：
  ```
  饭太硬: http://www.fantaiying.com/tv/
  巧记:   http://cdn.qiaoji8.com/tvbox.json
  肥猫:   http://wobushifengmao.love:63/
  ```

### 3. 设置播放器
- **硬解**：使用硬件解码（推荐，Mstar芯片支持）
- **软解**：使用软件解码（兼容性更好）

### 4. 保存并重启应用
- 点击 **保存设置**
- 重新打开应用，接口即生效

## 六、常见问题排查

### ❌ 安装失败："解析包错误"
**原因**：APK与系统版本不兼容
**解决**：确认电视系统为 Android 4.3+，重新下载兼容版本

### ❌ 安装失败："权限不足"
**原因**：未开启未知来源安装
**解决**：设置 → 安全 → 允许安装未知来源 → 开启

### ❌ 打开后黑屏
**原因**：接口源加载失败或网络问题
**解决**：
1. 检查网络连接
2. 进入设置 → 更换接口源
3. 确认接口地址可访问

### ❌ 播放卡顿
**原因**：网络带宽不足或解码方式不当
**解决**：
1. 切换为 **硬解** 模式
2. 降低播放分辨率
3. 检查网络速度（建议≥4M带宽）

### ❌ 遥控器无法操作
**原因**：焦点丢失或应用未响应
**解决**：按 **返回键** 退出，重新打开应用

## 七、进阶技巧

### 开机自启
应用已内置开机自启功能，安装后重启电视即可自动加载。

### 替换启动器
如果想让乐视TVBOX替代原装桌面：
1. 进入 **设置** → **应用管理**
2. 找到 **乐视TVBOX** → **设为默认桌面**

### 清除缓存
设置 → 应用管理 → 乐视TVBOX → 清除缓存
